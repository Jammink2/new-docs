Installing the Treasure Data CLI
================================

Install the [Treasure Data Toolbelt](http://toolbelt.treasuredata.com/) to set up your local workstation with `td`, the Treasure Data command-line client.

##Windows
You can download the installer from here.

- [Toolbelt Installer for Windows](http://toolbelt.treasuredata.com/win)


##Mac
You can download the installer from here.

- [Toolbelt Installer for Mac OS X](http://toolbelt.treasuredata.com/mac)

##Ubuntu (Precise or Lucid)
###Precise

Please add the Treasure Data apt packages repository to your sources list. This command will set it up for you:

	:::term
	$ curl -L http://toolbelt.treasuredata.com/sh/install-ubuntu-precise.sh | sh

###Lucid

Please add the Treasure Data apt packages repository to your sources list. This command will set it up for you:

	:::term
	$ curl -L http://toolbelt.treasuredata.com/sh/install-ubuntu-lucid.sh | sh

###init.d scripts

The /etc/init.d/td-agent script is used to start, stop, or restart the agent.

	:::term
	$ /etc/init.d/td-agent start
	$ /etc/init.d/td-agent stop
	$ /etc/init.d/td-agent restart

The logs are located under /var/log/td-agent. The daemon is running in td-agent user and td-agent group.

The config file is located at /etc/td-agent/td-agent.conf. Please restart the agent after whenever you have modified the file.

##RedHat/CentOS

This section shows how to install td-agent for CentOS, RHEL 5.0+.

First, please create **/etc/yum.repos.d/td.repo** with the following contents.

	:::text
	[treasuredata]
	name=TreasureData
	baseurl=http://packages.treasure-data.com/redhat/$basearch
	gpgcheck=0

Then, you can install td-agent via the yum command.

	:::term
	$ yum update
	$ yum install td-agent
	$ /etc/init.d/td-agent start

The */etc/init.d/td-agent* script is used to start, stop, or restart the agent.

	:::term
	$ /etc/init.d/td-agent start
	$ /etc/init.d/td-agent stop
	$ /etc/init.d/td-agent restart

The logs are located under */var/log/td-agent*. The daemon is running in *td-agent* user and *td-agent* group.

The config file is located at */etc/td-agent/td-agent.conf*. Please restart the agent after whenever you have modified the file.
