Installing the Treasure Data CLI
================================

Install the [Treasure Data Toolbelt](http://treasure-data.com/toolbelt.html) to set up your local workstation with the Treasure Data command-line client.

The Ruby command below will install the toolbelt:

    :::term
    $ gem install td

If you are not familiar with Ruby, alternative installation options can be found on the [Treasure Data Toolbelt page](http://treasure-data.com/toolbelt.html).
