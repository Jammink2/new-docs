# Treasure Data Android SDK

This article describes how to import data from Android applications.

## Overview

If you're building Mobile apps, you may want to import events/logs directly from the apps to Treasure Data. This can be achieved by embedding the [td-android-sdk](https://github.com/treasure-data/td-android-sdk) (Software Development Kit) into your applications.
