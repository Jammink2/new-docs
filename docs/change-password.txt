# Change the Password

This article describes how to change the password for the user.

## Prerequisites

  * Basic knowledge of Treasure Data, including [the toolbelt](http://toolbelt.treasure-data.com).
  * td command v0.10.43 or above

## Change Password

Please use `td user:password:change owner` command. Please always use `owner` as an argument for now.

    :::term
    $ td user:password:change owner
    New password (typing will be hidden): 
    Retype new password: 
    Password of user 'owner' changed.
