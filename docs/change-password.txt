# Change the Password

This article will describe how users can change their password for Treasure Data.

## Prerequisites

  * Basic knowledge of Treasure Data, including [the toolbelt](http://toolbelt.treasure-data.com).
  * td command v0.10.43 or above.

## Change Password

Please use the `td user:password:change owner` command. The `owner` argument is required for the time being.

    :::term
    $ td user:password:change owner
    New password (typing will be hidden): 
    Retype new password: 
    Password of user 'owner' changed.
