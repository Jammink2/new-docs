# Importing Data from Amazon Beanstalk Apps

By combining Amazon Beanstalk and Treasure Data, you can achieve both scalable frontend and backend system instantly.

## Prerequisites

  * Basic knowledge of Treasure Data
  * Basic knowledge of Amazon Beanstalk

## Installing td-agent

Please follow the instruction below, to install `td-agent`, our streaming data collector, into your application.

* [Installing td-agent on Amazon Beanstalk](http://help.treasure-data.com/kb/installing-td-agent-daemon/installing-td-agent-on-amazon-beanstalk)

## Import Data from Applications
