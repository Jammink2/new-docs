# Platform Constraints

Becuase Treasure Data is an cloud hosting environment, the platform has certain constraints that do not exist in a local Hadoop setup. These constraints ensure that your app takes advantage of Treasure Data's scalability and redundancy features.

## Hive User Defined Functions (UDF)

Custom Hive UDFs cannot be added. If you have a specific request for a function, please let us know.

## Hive Custom Map/Reduce Scripts

[Custom Map/Reduce scripts](https://cwiki.apache.org/confluence/display/Hive/Tutorial#Tutorial-Custommap%2Freducescripts) cannot be used due to security considerations.
