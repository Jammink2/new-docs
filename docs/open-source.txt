Open Source
===========

All of our client-side software are open-sourced. We have no black boxes in our client-side code.

# td command

The td command is comprised of three components. The source code repositories for each are shown below.

* [td](https://github.com/treasure-data/td)
* [td-client-ruby](https://github.com/treasure-data/td-client-ruby)
* [td-logger-ruby](https://github.com/treasure-data/td-logger-ruby)

# td-agent

td-agent consists of the Ruby interpreter, jemalloc library, and various Ruby gems. The source code repositories for each are shown below.

* [Ruby](http://www.ruby-lang.org/en/)
* [jemalloc](http://www.canonware.com/jemalloc/)
* [td-agent](https://github.com/treasure-data/td-agent) (Packaging Scripts)
* [fluentd](https://github.com/fluent/fluentd)
* [fluent-plugin-td](https://github.com/treasure-data/fluent-plugin-td)
