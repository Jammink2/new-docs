# Setup Consultation

We offer a paid `Setup Consultation Service`. This is usually 2 - 4 weeks process, heavily supported by Treasure Data engineers. The package includes:

* 1-on-1 call to describe the Basic Architecture of Treasure Data
* Weekly setup
* Setting up streaming or bulk data importers
* Setting up basic scheduled queries
* Setting up intermediate databases
* Setting up data visualization / dashboard / BI tools
